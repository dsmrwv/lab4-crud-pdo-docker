# Лабораторна №4: CRUD з PDO та Docker

Для кодування були використані лише стандартні можливості PHP 8.3 та офіційні Docker-образи:

- `php:8.3-fpm` – офіційний образ PHP
- `mysql:8.0` – офіційний образ MySQL
- `nginx:latest` – офіційний образ Nginx
- `phpmyadmin/phpmyadmin` – офіційний образ phpMyAdmin
